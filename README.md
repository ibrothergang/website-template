## SEO最友好的 html 代码结构

- title 写清楚你要优化的词
- description 介绍
- h1 再次写清楚你要优化的词

h2 二级标题1
p 二级标题1下的介绍性内容

h3 二级标题1下的三级标题1
p 二级标题1下的三级标题1下的介绍性内容

h3 二级标题1下的三级标题2
p 二级标题1下的三级标题2下的介绍性内容

h2 二级标题2
p 二级标题2下的介绍性内容

h3 二级标题2下的三级标题1
p 二级标题2下的三级标题1下的介绍性内容

h3 二级标题2下的三级标题2
p 二级标题2下的三级标题2下的介绍性内容
